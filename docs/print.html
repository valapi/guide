About

valorant.ts is a NodeJS package that make more easier to use Valorant API

  • All-in-one package
  • Support Multi-factor Authentication (2FA)
  • 5 Packages:
    • auth
    • lib
    • riot-api
    • valorant-api.com
    • web-client

NPM:

npm install valorant.ts

Yarn:

yarn add valorant.ts

PNPM:

pnpm add valorant.ts

  • Node.js (we recommend lastest version for best performance)

Basic Authentication

Login

First We Need To Login To Valorant Account

await AuthClient.login('Username', 'Password');

But We Have 2 Type Of Account

Multi-Factor

When The Account Have Multifactor You Can't Use Normal Method

After Login The Verify Code Will Send To Mail

await AuthClient.verify(SixDigitNumber);

Advanced Authentication

Reconnection

its will reconnection if the token is expired

we recommend running this after creating AuthClient

await AuthClient.refresh(isForce?);

Cookie

you can save only SSID cookie for ~30 days.

const AuthClient = await ValAuth.fromCookie(cookie.ssid);

Language

valorant-api.com has a specific language called all, that will return all languages.

{
  'ar-AE': 'فايد',
  'en-US': 'Fade',
  'ja-JP': 'フェイド',
  'ko-KR': '페이드',
  'zh-TW': '菲德'
}

but if you aren't using all language, you can use en-US, th-TH, and more, but it will return only one language you choose.

'フェイド' // ja-JP (japanese)

Typescript

the language will not affect with javascript user. for typescript users, we recommend using all language.

// const MyRequest = await ApiClient.Weapons.get();

const Data: string = MyRequest.data.data[0].displayName['ko-KR']

const Datas: Array<string> = [

    MyRequest.data.data[2].displayName['en-US'],
    MyRequest.data.data.at(3).displayName['ja-JP'],

];

but when you want to, we recommend doing this:

const NotError: any = MyRequest.data.data[0].displayName;

const TypesError: string = MyRequest.data.data[0].displayName;
