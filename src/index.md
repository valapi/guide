# About

---

`valorant.ts` is a NodeJS package that make more easier to use Valorant API

-   All-in-one:
    -   Authentication
        -   Two-Factor Authentication
        -   Riot Identity
        -   CAPTCHAs
    -   Crosshair Serializer
    -   Official API
        -   Leaderboard
    -   valorant-api.com
        -   ID
        -   Information
    -   Web Client
        -   Daily Store
        -   Personal Data
-   Open-source
-   Typescript Support
-   Built-In API Endpoints
