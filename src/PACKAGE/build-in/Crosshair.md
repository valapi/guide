# Crosshair Compiler

> This feature is in Pre-alpha

---

## Client

```typescript
import { ValCrosshair } from "valorant.ts";
```

```typescript
const myCrosshair = new ValCrosshair();
```

## Usage

Import

```typescript
myCrosshair.import(profileCode);
```

Export

```typescript
myCrosshair.export(); // => Profile Code
```
